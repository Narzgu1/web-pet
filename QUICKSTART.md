# Быстрый старт

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Приложение откроется на `http://localhost:5173`

## Сборка

```bash
npm run build
```

Собранные файлы будут в папке `dist/`

## Структура

- `src/pages/` - все страницы приложения
- `src/components/` - переиспользуемые компоненты (Navbar, Footer, Modal)
- `src/data/` - мок-данные в JSON формате
- `src/lib/` - утилиты (форматирование дат, localStorage)
- `src/styles/` - глобальные стили Tailwind

## Особенности

✅ Компонентная архитектура  
✅ Единый Navbar и Footer  
✅ React Router для навигации  
✅ Мок-данные готовы к замене на API  
✅ Адаптивный дизайн  
✅ Готово к расширению
