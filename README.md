# PetCareHub

Веб-приложение для сообщества владельцев домашних животных, разработанное на стеке Vite, React и Tailwind CSS.

## Технологии

* **Vite** — инструмент сборки и среда разработки.
* **React** — JavaScript-библиотека для создания пользовательских интерфейсов.
* **React Router** — библиотека для декларативной маршрутизации.
* **Tailwind CSS** — фреймворк для стилизации на основе утилитных классов.
* **Font Awesome** — библиотека иконок.
## Структура проекта

```text
pet-web/
├── public/
│   ├── images/          # Статические изображения и аватары
│   └── favicon.svg      # Иконка приложения
├── src/
│   ├── pages/           # Основные страницы (Home, Forum, Admin и др.)
│   ├── components/      # Переиспользуемые UI-компоненты (Navbar, Modal)
│   ├── data/            # Локальные JSON-файлы с мок-данными
│   ├── styles/          # Глобальные стили и конфигурация Tailwind
│   ├── lib/             # Вспомогательные утилиты и функции
│   ├── App.jsx          # Корневой компонент с роутингом
│   └── main.jsx         # Точка входа в приложение
├── docs/                # Сопутствующая документация
└── screenshots/         # Материалы для демонстрации проекта
```
Установка и запуск
Для запуска проекта на локальной машине выполните следующие команды:
# Установка зависимостей
npm install

# Запуск сервера для разработки
npm run dev

# Сборка проекта для продакшена
npm run build

# Предпросмотр собранной версии
npm run preview
