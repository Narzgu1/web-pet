# PetCareHub

Веб-приложение для сообщества владельцев домашних животных, разработанное на стеке Vite, React и Tailwind CSS.

## Технологии

* **Vite** — инструмент сборки и среда разработки.
* **React** — JavaScript-библиотека для создания пользовательских интерфейсов.
* **React Router** — библиотека для декларативной маршрутизации.
* **Tailwind CSS** — фреймворк для стилизации на основе утилитных классов.
* **Font Awesome** — библиотека иконок.
