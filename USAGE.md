# Инструкция по использованию

## Быстрый старт

1. **Установите зависимости:**
   ```bash
   npm install
   ```

2. **Запустите dev-сервер:**
   ```bash
   npm run dev
   ```

3. **Откройте браузер:**
   - Приложение доступно на `http://localhost:5173`

## Тестирование функций

### 1. Авторизация

**Вход как администратор:**
- Email: `admin@petcarehub.com`
- Password: `admin123`
- Доступ: все страницы, включая Admin Panel

**Вход как пользователь:**
- Email: `user@test.com`
- Password: `user123`
- Доступ: все страницы кроме Admin Panel

**Регистрация нового пользователя:**
- Нажмите "Register Now" или "Login" в Navbar
- Переключитесь на вкладку "Register"
- Заполните форму (имя, email, пароль)
- По умолчанию создается пользователь с ролью `user`

### 2. Forum

**Создание поста:**
1. Войдите в систему
2. Перейдите на страницу Forum
3. Нажмите "Start New Discussion"
4. Заполните форму (заголовок, категория, текст)
5. Нажмите "Post"

**Поиск и фильтрация:**
- Используйте поле поиска для поиска по тексту
- Выберите категорию в выпадающем списке
- Кликните на категорию в боковой панели

**Просмотр поста:**
- Кликните на карточку поста для просмотра полного текста

### 3. Lost & Found

**Создание объявления:**
1. Заполните форму (имя, статус, локация, описание, контакты)
2. Загрузите фото (опционально)
3. Нажмите "Submit Report"

**Фильтрация:**
- Используйте кнопки "All", "Lost", "Found" для фильтрации

**Просмотр деталей:**
- Кликните на карточку для просмотра полной информации

### 4. Pet Profile

**Редактирование профиля:**
1. Войдите в систему (требуется авторизация)
2. Выберите питомца в боковой панели
3. Нажмите "Edit Profile"
4. Измените данные и нажмите "Save"

**Загрузка аватара:**
1. Нажмите "Change Avatar"
2. Выберите изображение
3. Аватар обновится автоматически

**Добавление визита к ветеринару:**
1. Нажмите "Add Visit" в разделе "Vet Visits History"
2. Заполните форму (дата, причина, заметки)
3. Нажмите "Add Visit"

**Удаление визита:**
- Нажмите "Delete" на карточке визита

**Статус вакцинации:**
- Используйте переключатель для изменения статуса

### 5. Facilities Map

**Использование карты:**
1. Используйте фильтры слева для выбора типов заведений
2. Кликните на маркер на карте
3. Откроется popup с краткой информацией
4. На десктопе информация также отображается в правой панели
5. Кликните на маркер для открытия модалки с деталями

**Фильтрация:**
- Отметьте/снимите галочки для типов заведений
- Нажмите "Apply Filters"

### 6. Admin Panel

**Доступ:**
- Только для пользователей с ролью `admin`
- Автоматический редирект, если вы не админ

**Модерация постов:**
1. Перейдите на вкладку "Forum Posts"
2. Просмотрите список постов
3. Нажмите "Approve" для одобрения
4. Нажмите "Delete" для удаления

**Модерация объявлений:**
1. Перейдите на вкладку "Lost & Found"
2. Просмотрите список объявлений
3. Нажмите "Approve" или "Delete"

**Статистика:**
1. Перейдите на вкладку "Statistics"
2. Просмотрите общую статистику:
   - Количество пользователей
   - Количество постов (всего / одобренных)
   - Количество объявлений (всего / одобренных)

## Важные замечания

⚠️ **Все данные хранятся в localStorage браузера:**
- При очистке данных браузера все данные будут потеряны
- Данные не синхронизируются между устройствами
- Это демо-версия для диплома, в продакшене нужен бэкенд

⚠️ **Безопасность:**
- Пароли хранятся в открытом виде (для демо)
- В реальном приложении нужна хеширование паролей
- Нужна защита от XSS и CSRF

## Структура данных

Все данные можно просмотреть в DevTools → Application → Local Storage
